# Metro Station Console Application

This is a simple console application that allows users to input metro station data. Users can input the metro station name, ID, number of tickets sold, and the metro line color.

## Installation

1. Clone the repository to your local machine.
2. Make sure you have a compatible C# compiler installed.
3. Open the solution file in your preferred integrated development environment (IDE).

## Usage

1. Run the program.
2. Follow the prompts to input the metro station data.
